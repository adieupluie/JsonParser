# JSON Parser Viewer UI 와이어프레임

## 1. 메인 레이아웃

- 전체 창: 3단 분할
  - 좌측: JSON 에디터 (40%)
  - 중앙: 트리 뷰 (40%)
  - 우측: 노드 인스펙터 (20%)
- 하단 상태바: 파싱 결과, 노드 수, 선택 경로, 시간

## 2. 상단 툴바 요소

- 버튼
  - Open
  - Save
  - Parse
  - Validate
  - Pretty Print
  - Expand All
  - Collapse All
  - Export Tree
- 검색창
  - 노드/값/경로 검색

## 3. JSON 에디터 패널

- 텍스트 입력 영역
- 라인 번호 영역
- `Parse on Type` 토글
- 오류 라인 마커
- 입력 텍스트 배경: 밝은 회색
- 오류 강조: 붉은 밑줄 + 라인 헤더 색 변경

## 4. 트리 뷰 패널

- 노드 형태
  - 객체: 폴더 아이콘 + `{}`
  - 배열: 리스트 아이콘 + `[]`
  - 원시: 값 아이콘 + 텍스트
- 노드 항목 구성
  - 키
  - 타입/값 축약
  - 확장/축소 버튼
- 노드 선택 스타일
  - 배경 강조
  - 우측 인스펙터에 상세 표시

## 5. 노드 인스펙터 패널

- 필드
  - Path
  - Type
  - Value
- Copy 버튼
  - Copy Path
  - Copy Value
- 편집 버튼
  - Apply
- 메타/주석 표시 영역 (옵션)

## 6. 상태바

- 파싱 상태 텍스트
- 오류개수/성공 여부
- 선택 경로 요약
- 파싱 시간

## 7. 상호작용 시나리오

### 7.1 정상 파싱

- 사용자가 JSON 입력 후 `Parse` 실행
- 트리 뷰에 계층 표시
- 상태바에 `Parsed successfully` 표시

### 7.2 오류 발생

- 에디터에 오류 위치 하이라이트
- 상태바/패널에 오류 메시지 표시
- `Go to Error` 이동 기능

### 7.3 노드 선택

- 트리에서 노드 클릭 → 인스펙터에 정보 로드
- `Copy Path` 및 `Copy Value` 사용 가능
- `Apply`로 수정 사항 반영

### 7.4 검색

- 검색어 입력 시 트리에서 일치 항목 표시
- 결과 목록으로 빠른 이동

## 8. 접근성 및 키보드 플로우

- Tab 키로 패널 간 이동
- 화살표 키로 트리 탐색
- Enter: 선택/확장
- Ctrl+P: Parse
- Ctrl+F: Search
- Ctrl+E / Ctrl+Shift+E: Expand/Collapse
- F8: 다음 오류로 이동

## 9. UI 스타일 가이드 요약

- 배경: `#FFFFFF`
- 경계: `#E6E6E6`
- 텍스트: `#111111`
- 액센트: `#1890FF`
- 오류: `#FF4D4F`
- 노드 hover: `#F5F7FA`
- 본문 글꼴: 14px 산세리프
- 코드/JSON 에디터: 13px 모노스페이스
